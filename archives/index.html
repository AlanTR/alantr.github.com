考研复习了，写点什么吧

已经算开始准备考研了，可是整天还是浑浑噩噩的。前几天草草定的计划到现在都没有正经实施过。呃，我在磨叽什么呢。。。。。？

假期里兴起了搞个自己小站的意愿后，废寝忘食的半个周，才把这里倒腾起来。（没学过js还有网站建设的课程也没有好好听的我压力好大啊 ╭(╯^╰)╮）

假期里写的英雄联盟挂机辅助的辅助补丁（好像有点拗口）老早就给改成了可以随着那个外挂的升级而进行自我更新的版本（每次它升级都要亲自改代码实在 太麻烦了），只要有个性能稳定的计算机就可以无限的用下去。不过半个月前就已经不用了。想来有些好笑：当初为了玩这个游戏，不惜花那么多精力在上面， 连参加软件设计大赛也没花过这些功夫。可是现在连打开游戏的动力都没有了。

为了一个小小的游戏辅助程序，可以忍受着家里1M接入的网速在半个月内学习完全没有接触过的知识，用之做出成品，并在接下来的半个月内测试完善出成品， 而软件设计大赛里我发挥的作用却微乎其微？……似乎我大一的时候就该反省一下自己。

晕，糊里糊涂的扯的这么远。

我还真不知道为什么会扯到这里的。可是最近实在是觉得，自己要反省的东西太多了。

Windows平台下git中文乱码的问题

进入git安装目录，改一下配置就可以基本解决： 1、etc\gitconfig：

[gui] encoding = utf-8 [i18n] commitencoding = gbk [svn] pathnameencoding = gbk

说明：打开 Git 环境中的中文支持。pathnameencoding设置了文件路径的中文支持。
