Windows平台下git中文乱码的问题

进入git安装目录，改一下配置就可以基本解决： 1、etc\gitconfig：

[gui] encoding = utf-8 [i18n] commitencoding = gbk [svn] pathnameencoding = gbk

说明：打开 Git 环境中的中文支持。pathnameencoding设置了文件路径的中文支持。
